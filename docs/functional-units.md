# Functional Units
A informatory text to tell you what type of units EandoX uses and does not use. Aswell as what you can use them for.

### Units in use

1.KG-Kilogram
2.PC-Parsec
3.M-Meter
4.M2-Square Meter
5.m3-Cubic meter

## Other
EandoX does currently not support the Imperial/US-customary system.
